# Portfolio Website

This is a personal portfolio website built with React and Bootstrap.

## Tech Stack

- React.js
- React-Bootstrap
- CSS3
- React Router

## Features

- Multi-page layout (`Home`, `About`, `Projects`, `Resume`, `Blog`)
- Responsive design
- Project cards with GitHub/demo links
- Resume preview and download
- GitHub contribution calendar

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm start
```

3. Open:

`http://localhost:3000`

## Customize Content

Edit files under `src/components/` to update personal information, projects, and links.
